Création d'un site web

Description du projet

Ce projet est mon premier projet universitaire réalisé durant ma première année à l'IUT. L'objectif était de créer un site web sur le sujet de notre choix avec une contrainte particulière : sans utiliser de div ni de flexbox. Cette limitation a poussé à utiliser des approches CSS alternatives et créatives.

Objectifs

  • Créer un site web fonctionnel et responsive
  • Maîtriser HTML et CSS sans outils modernes (pas de flex)
  • Mettre en place une navigation fluide
  • Gérer les images et l'intégration responsive

Technologies utilisées

HTML5 CSS3

Points forts

  • Compréhension approfondie du CSS positionnement
  • Gestion responsive sans flexbox
  • Code structuré et respectant les bonnes pratiques de HTML

Apprentissages

Ce projet m'a permis de comprendre les fondamentaux du CSS et du positionnement. En étant forcé à ne pas utiliser flexbox, j'ai dû apprendre les alternatives comme le positionnement absolu, le float et l'utilisation intelligente des propriétés display.
