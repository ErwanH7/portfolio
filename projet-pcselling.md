            <h3>Description du projet</h3>
            <p class="project-description">
                Ce projet universitaire m'a permis de développer un véritable site web en utilisant <strong>PHP</strong> et une 
                <strong>base de données SQL</strong>. L'objectif était de créer une plateforme fonctionnelle avec gestion des données, 
                authentification utilisateur et interface web interactive.
            </p>

            <h3>Objectifs</h3>
            <ul class="objectives-list">
                <li>Maîtriser PHP et ses concepts de programmation serveur</li>
                <li>Concevoir et utiliser une base de données SQL</li>
                <li>Créer une interface web dynamique</li>
                <li>Gérer l'authentification et les sessions utilisateur</li>
                <li>Implémenter des fonctionnalités CRUD (Create, Read, Update, Delete)</li>
                <li>Déployer le site en ligne</li>
            </ul>

            <h3>Technologies utilisées</h3>

            <h3>Fonctionnalités implémentées</h3>
            <ul class="features-list">
                <li>Connexion et authentification utilisateur</li>
                <li>Gestion des profils utilisateurs</li>
                <li>Opérations CRUD sur la base de données</li>
                <li>Validation des formulaires côté serveur</li>
                <li>Gestion des sessions</li>
                <li>Interface responsive et intuitive</li>
